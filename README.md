# The Seer
A C# Console app that handles your Tarot, Oracle and Rune readings.
### Features

- Let's the user create their own account for saving data as readings or journal entries
- Can use several decks, different types of deck systems, individual spreads
- Database connected for lasting data
- A library to study the different cards of each deck
- Sleek and fun interface

### Installation
- See the latest release for installation and setup

### Usage
- Either double-click the `TheSeer.exe` file or run it from the command line.
- Follow the on-screen prompts to receive your card reading.
- Enjoy your reading!
